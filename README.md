# British Airways Booking Prediction

End-to-end data science project developed as part of the British Airways Data Science Virtual Experience.

## Project Overview

This project predicts the probability that a customer will complete a flight booking using historical booking data.

The workflow includes:
- Data preparation and feature engineering in JMP Pro
- Predictive modelling in JMP Pro and Python
- Business visualisation in Tableau
- Model comparison and business recommendations
